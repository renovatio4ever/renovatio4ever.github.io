NODE JS LIRI BOT


Online User Guide

Welcome to the Node JS Liri Bot. This is a simple Node JS solution that process user inputs and provides responses based on a preselected list of functions. The following is a list of instructions to install, configure and run the Liri Bot.

Github: Node JS Liri Bot

Implmentation Guide

    Step 1: Install Node JS

  • Visit node.js and down load the latest build
  • https://nodejs.org/en/

 

    Step 2: Launch the IDE

  • After downloading and installing Node JS, Open a terminal session

 

    Step 3: Clone/Copy the repository

  • Visit my Liri-Node-App repository and copy the contents to your IDE

    •  
 

    Step 4: Obtain Spotify Key

  • You will need to register and obtain both a client and secret key. Create a .env file in the project and add your client, and secret key to the file.

 

    Step 5: Obtaining NPM Packages

  • The solution relies on a few node management packages (NPM) packages for functionality. These packages can be obtain from NPM along with guides on how to use each. For this project, only an installation is required.

 

    Step 6: Installing the require NPM Packages

  • The solution relies on a few npm packages for functionality. In the terminal install the following packages

 

    The following npm are api's to the required data source for Band Concerts, Spotify Music, and Movies

  • npm install bandsintown
  • npm install node-spotifiy-api
  • npm install omdb
    •  

    The following npm are required for user requests, and the passing of secret keys to Spotify

  • npm install axios
  • npm install dotenv
    •  

    The following npm are required for the solution to write to a log file, date and time, and accept user and system commands

  • npm install fs
  • npm install moment
  • npm install node-cmd
    •  

    The following npm packages are for cosmetic affects including color and large ascii text

  • npm install chalk
  • npm install figlet

 

    Step 7: Run the Liri Bot

    The following commands are available for the liri bot

  • "node liri.js help Presents solutions how-to"
  • "node liri.js concert-this name of band"
  • "node liri.js spotify-this-song Presents solutions how-to"
  • "node liri.js movie-this name of movie"
  • "node liri.js do-what-it-says"
