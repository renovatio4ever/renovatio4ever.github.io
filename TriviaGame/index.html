Video Game Trivia

Welcome! This is a simple trivia game based on the Ready Player One them set. The rules are simple: Pick the correct answer within the time provided. The answer will be provided if the time expires. Be warned however, answered and unanswered questions will be tallied. Object of the game is to try to get them all answered correctly The game features the application of HTML, HTML5, CSS, Jquery, JavaScript, Bootstrap, Sound, and gif animation. Click on the x icon to the upper right close these instructions. Remember to turn up the volume, and Good Luck Player!
