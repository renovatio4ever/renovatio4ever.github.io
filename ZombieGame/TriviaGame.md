# Assignment

A simple zombie game

# Rules

Guess the number the zombie is thinking of. Incorrect guesses will result in damages inflicted upon you. 
Correct guesses inflict damage on the zombie. First player to reach zero losses.

## Technologies

The game was developed in Javascript, NodeJS. Uses the inquiry package for guesses, and colors for 
effects!
